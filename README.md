## 👨🏽‍💻 Tech Stack

| What?             | How?           |
| :---------------- | :------------- |
| State management  | Bloc / Cubit   |
| Api service       | Dio / Retrofit |
| Offline database  | Sqlite         |
| Push notification | Firebase       |

## ☑️ Features

- [ ] search weather information for a city
- [ ] a 3-day weather forecast
- [ ] favorite cities for offline access
- [ ] push notifications
